# 升級規則說明

## 等級範圍

最低等級為 **LV1**，最高等級為 **LV120**。

## 升級所需任務數

| 等級區間 | 每升一級所需完成任務數 | 累計任務數 |
| -------- | ---------------------- | ---------- |
| LV1 → LV5 | 3 個任務 | 0 ~ 15 |
| LV6 → LV10 | 5 個任務 | 15 ~ 40 |
| LV11 → LV120 | 10 個任務 | 40 ~ 1140 |

### 計算範例

- 完成 **0** 個任務 → LV1
- 完成 **3** 個任務 → LV2
- 完成 **15** 個任務 → LV6
- 完成 **40** 個任務 → LV10（進入第二階段）
- 完成 **50** 個任務 → LV11
- 完成 **100** 個任務 → LV16

## 階段系統

角色根據等級自動進入不同階段，每個階段可自訂大頭貼與職業名稱：

| 階段 | 等級範圍 | 預設職業名稱 |
| ---- | -------- | ------------ |
| 1 | LV1 ~ LV10 | 初心者 |
| 2 | LV10 ~ LV30 | 見習戰士 |
| 3 | LV30 ~ LV70 | 冒險者 |
| 4 | LV70 ~ LV120 | 英雄 |

## 核心任務

- 每次只能標記一個任務為「核心任務」（星號標記）
- 核心任務完成時會更新核心任務進度條

## 編輯模式

進入編輯模式（右上角按鈕）後，可以：

- 在「階段設置」頁籤中修改每個階段的大頭貼與職業名稱
- 修改角色大頭貼與圖片位置
